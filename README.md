# kadaberice 🌌

My personal **Arch Linux + Hyprland** desktop configuration ("rice") and development environment setup.

## 🚀 Features & Components

- **Window Manager:** [Hyprland](https://hyprland.org/) (Dynamic tiling Wayland compositor)
- **Editor:** Professional [LazyVim](https://www.lazyvim.org/) setup optimized for C, C++, Rust, Python, Go, and Web Development.
- **Shell & Tools:** Bash, fast screenshot workflow (`grim` + `slurp` + `swappy`).

## 📁 Repository Structure

* `.config/hypr/` — Hyprland main desktop configurations and keybindings.
* `.config/hypr/scripts/` — Custom desktop productivity utilities (e.g., custom screenshot script).
* `.config/nvim/` — Custom, modular, high-performance Neovim IDE configurations.
